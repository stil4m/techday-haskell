Least Divisor

Given i have $n$ and I want to find its least divisor

  1. Let $k = 2$
  2. Check if $k$ divides $n$, then return $k$
  3. Check if $k$ exceeds the square root of $n$, then return $n$
  4. Otherwise go back to (2) with $k+1$
ld :: Integer -> Integer
ld n = ldf 2 n

ldf k n | k `divides` n = k
        | k^2 > n = n
        | otherwise = ldf (k+1) n

Pattern Matching

data Expr = I Int
          | Add Expr Expr
          | Mult Expr Expr
          deriving (Show)
expr1 = (Mult (I 4)
              (Add (I 1) (I 2)))
expr2 = (Add (I 4)
             (Mult (I 1) (I 2)))
eval :: Expr -> Int
eval (I i) = i
eval (Add l r) = eval l + eval r
eval (Mult l r) = eval l * eval r

Composition in Haskell

1) $h(x) = g(f(x))$

f = (+1)
g = show

h x = g (f x)

2) $h(x) = g(f(x))$

f = (+1)
g = show

h x = g $ f x

3) $h = g \cdot f$

f = (+1)
g = show

h = g . f
